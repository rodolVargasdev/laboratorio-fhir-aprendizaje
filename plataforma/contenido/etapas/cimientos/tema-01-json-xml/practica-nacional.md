## Tema 1 · JSON y XML → El paciente salvadoreño en FHIR

- Práctica (45 min, PC): escribe a mano un `Patient` JSON válido para un paciente
  ficticio salvadoreño: nombre con dos apellidos (`family` más la convención que
  decidas documentar), `identifier` con DUI ficticio (inventa un `system` nacional
  tipo `https://salud.gob.sv/fhir/identificadores/dui`) y otro con número de expediente,
  teléfono +503, dirección con departamento y municipio.
- Entregable: `institucion/tema-01/paciente-sv.json` + notas de las decisiones que tomaste.
- Esto demuestra: sabes mapear la identidad del paciente salvadoreño al modelo FHIR.
