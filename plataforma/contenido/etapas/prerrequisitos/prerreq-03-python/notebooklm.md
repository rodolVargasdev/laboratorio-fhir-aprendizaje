# NotebookLM — Python para interoperabilidad

> Paso obligatorio para consolidar el tema antes del quiz.

Crea un cuaderno en [NotebookLM](https://notebooklm.google.com) con el nombre y las fuentes que la app te muestra en este tema (los toma de `tema.json`): el tutorial oficial de Python en español, el módulo `json` y la documentación de `requests`. Agrega también la lección de este tema como fuente propia. Luego usa los tres prompts sugeridos por la app: uno para contrastar corchetes contra `.get()`, otro para practicar diagnóstico de errores por traceback y un examen oral de 8 preguntas sobre `json`, `requests` y la estructura de un Bundle. Genera el Audio Overview y escúchalo antes de intentar el quiz.
